# Запуск

```shell
docker run -d \
        --name postgres_sirius_2024 \
        -e POSTGRES_USER=sirius_2024 \
        -e POSTGRES_PASSWORD=change_me \
        -p 38746:5432 \
        postgres:15.5
```
